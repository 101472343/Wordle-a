Instructions:

1) Open Powershell Terminal

2) Create directory and change directory by typing as follows:
- to make directory type: mkdir "Folder Name"
- to change to that directory, type: cd "Folder Name" 

3) Clone the Repository -
Open your terminal or command prompt and run the following command to clone the Wordle-anche! repository:

git clone https://github.com/sabazubair/wordle-anche.git

4) Navigate to the Project Directory - 
Change your working directory to the newly cloned Wordle-anche! folder:

cd wordle-anche

5) Install Dependencies -
Run the following command to install the necessary dependencies:

npm install

6) Run the Application - 
Once the installation is complete, start the Wordle-anche! application:

npm start

This command will start the application, and you should be able to access it by opening your web browser and navigating to http://localhost:3000/ (or another port if specified).
